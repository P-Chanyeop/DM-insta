## 2. 한국 시장 현황

### 핵심 수치
- **인스타그램 한국 MAU:** 2,644만 (한국 SNS 앱 1위)
- **앱 사용률:** 78.8% (평균 53.8% 대비 25% 높음)
- **글로벌 마케팅 자동화 시장:** 2025년 $72.3억 → 2034년 $201.2억 (CAGR 12%)

### 한국 사용자가 가장 필요로 하는 기능 (우선순위)
1. 댓글→DM 자동화 ("신청", "구매", "가격" 키워드)
2. 공동구매 운영 지원 (신청/알림/추첨)
3. FAQ 자동 응답 (영업시간, 배송, 가격)
4. 한국어 AI 자동 응답
5. 스토리 멘션/반응 자동 답장
6. 스마트스토어/쿠팡/네이버페이 연동
7. 캐러셀 메시지 (제품 카탈로그 DM)

### Instagram API 제한사항
- 시간당 200 DM 발송 제한
- 24시간 메시징 윈도우 (사용자 인터랙션 후)
- 댓글당 1개 Private Reply (7일 이내)
- Meta 공식 API만 허용 (비공식 도구 = 계정 정지)

## 4. 경쟁 우위 전략: 센드잇만의 차별점

### 전략 1: "한국형 인스타 자동화" 포지셔닝

ManyChat은 영어 중심이고, 소셜비즈는 기능이 제한적입니다.  
센드잇은 **한국 비즈니스 패턴에 최적화된 깊은 자동화**를 제공합니다.

**구체적 실행:**
- 공동구매 전용 템플릿 ("신청" 댓글 → DM → 폼 수집 → 당첨 발표)
- 스마트스토어/쿠팡 주문 상태 자동 DM 알림
- 한국어 AI 응답 (브랜드 톤 커스터마이징)
- 업종별 원클릭 설정 (의류/뷰티/F&B/교육/PT)

### 전략 2: 투명한 가격 정책

ManyChat의 최대 불만은 **컨택 기반 과금의 예측 불가능성**입니다.

**추천 가격 구조:**

| 플랜 | 월 가격 | 포함 내용 |
|------|--------|----------|
| **무료** | 0원 | 월 500 DM, 1개 플로우, 센드잇 브랜딩 |
| **스타터** | 9,900원 | 월 3,000 DM, 5개 플로우, 브랜딩 제거 |
| **프로** | 29,900원 | 무제한 DM, 무제한 플로우, AI 응답, 분석 |
| **비즈니스** | 79,900원 | 프로 + 팀 협업, API 접근, 우선 지원 |

> **핵심:** 메시지 수 기반 과금 (컨택 수 아님). 스팸 봇이 DM 보내도 요금 폭탄 없음.

### 전략 3: 5분 설정 → 즉시 효과

소셜비즈 10분, 키티챗 1분을 벤치마크합니다.

**구체적 실행:**
- Instagram 계정 연결 → 업종 선택 → 추천 템플릿 자동 적용 → 완료
- 첫 플로우까지 5스텝 이내
- 온보딩 완료 시 실제 동작하는 데모 플로우 자동 생성

### 전략 4: 경쟁사에 없는 킬러 피처

| 피처 | 설명 | 경쟁사 현황 |
|------|------|-----------|
| **스마트 댓글 답글** | 댓글에 자동 공개 답글 + DM 동시 발송 | ManyChat만 유사 기능 |
| **공동구매 모드** | 신청→수량확인→결제→배송알림 전체 자동화 | 와이어디만 일부 지원 |
| **AI 대화** | GPT 기반 한국어 FAQ + 맥락 유지 | 키티챗만 기본 수준 |
| **실시간 분석 대시보드** | 노드별 전환율, 이탈 지점 시각화 | 모든 경쟁사 약함 |
| **카카오톡 연동** | IG DM + 카카오톡 채널 통합 자동화 | 어떤 경쟁사도 미지원 |

## 6. 결론: 센드잇의 승리 공식

```
한국어 네이티브 UX
+ 공동구매/커머스 특화
+ 투명한 메시지 기반 과금
+ 5분 설정
+ AI 한국어 응답
= ManyChat이 절대 못 따라오는 한국 시장 전용 제품
```

**핵심 메시지:** ManyChat을 따라가지 말고, 한국 인스타 셀러가 "이건 나를 위해 만든 서비스다"라고 느끼게 만들어야 합니다.

소셜비즈는 가볍고, 와이어디는 커머스에 치중하고, ManyChat은 한국어를 모릅니다.  
센드잇은 **깊은 자동화 + 한국 커머스 + AI**를 결합한 유일한 포지션을 차지할 수 있습니다.
