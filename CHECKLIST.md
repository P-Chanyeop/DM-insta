## 사용자가 직접 해야 하는 작업 (⚠️ 항목)

### 필수: Meta (Instagram) 연동
- [ ] [Meta for Developers](https://developers.facebook.com/)에서 앱 생성
- [ ] Instagram Graph API 권한 추가: `instagram_basic`, `instagram_manage_messages`, `instagram_manage_comments`, `pages_messaging`
- [ ] Instagram 비즈니스 계정 → Facebook 페이지 연결
- [ ] OAuth Redirect URI: `https://your-domain.com/api/auth/instagram/callback`
- [ ] 앱 ID / App Secret → 설정 > 연동(API) > Instagram Graph API에 입력
- [ ] Webhook URL: `https://your-domain.com/api/webhook/instagram`
- [ ] 앱 검수 제출 (Live 모드 전환)

### 선택: 외부 서비스 연동
- [ ] **OpenAI**: API Key 발급 → 설정 > 연동(API) > OpenAI/ChatGPT
- [ ] **Shopify**: Partners Dashboard에서 앱 생성 → API Key/Secret 입력
- [ ] **Google Sheets**: Cloud Console > Sheets API > 서비스 계정 키
- [ ] **Stripe**: Publishable Key + Secret Key 입력
- [ ] **Klaviyo**: Private API Key 입력

### 배포
- [ ] 도메인 & SSL 설정
- [ ] DB: H2 → MySQL/PostgreSQL 전환
- [ ] `application.yml`: DB URL, JWT Secret 변경
- [ ] `.env.production`: `VITE_API_BASE_URL` 설정
- [ ] `cd frontend && npm run build`
- [ ] `cd backend && ./gradlew bootJar`
